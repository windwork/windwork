Windwork 性能分析类
======================
记录程序执行时间（秒）、内存占用（M）。

标记点读取
```
$mark = \wf\core\Benchmark::mark('标记_ID'); // 标记_ID不能重复，否则为直接读取上一个标记点的值

print_r($mark);

/*
Array (
  [time] => 1492398034.4373 // 程序在当前标记点执行时间（秒）
  [mem] => 0.3726           // 程序在当前标记点占用内存（M）
  [elapsed] => 0.0007       // 程序从开始执行到当前标记点所用时间（秒）
)
*/

```
